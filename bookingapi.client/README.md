# Booking System

A full-stack booking system built as a side project to explore C# and REST API development.

## Tech Stack

**Backend**

- C# / ASP.NET Core Web API (.NET 10)
- Entity Framework Core
- SQLite

**Frontend**

- React 19
- TypeScript
- Vite

## Features (in progress)

- [ ] Create and manage bookings
- [ ] Customer management
- [ ] Service management
- [ ] Admin dashboard
